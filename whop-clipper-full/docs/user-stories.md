# WhopClipper - User Stories (Updated)

**Product:** WhopClipper  
**Date:** 2026-03-09

---

## Two Types of Users

### User Type A: Campaign Creators (Whop Businesses)
- Create clipping campaigns
- Pay clippers for content
- Track performance

### User Type B: Clippers (Content Creators)
- Find campaigns to work on
- Clip videos
- Get paid for views

---

## Campaign Creator Stories

### CC-1: Create Campaign
- **As a** Whop business owner,
- **I want to** create a clipping campaign with my YouTube videos
- **So that** I can pay creators to make short-form content from my content

### CC-2: Set Budget
- **As a** Whop business owner,
- **I want to** set a budget and payment terms (per view, capped)
- **So that** I can control costs and reward top performers

### CC-3: Review Submissions
- **As a** Whop business owner,
- **I want to** review submitted clips
- **So that** I can approve quality content

### CC-4: Track Performance
- **As a** Whop business owner,
- **I want to** see views and engagement on submitted clips
- **So that** I can measure ROI

---

## Clipper Stories

### CP-1: Find Campaigns
- **As a** clipper,
- **I want to** browse available clipping campaigns
- **So that** I can find ones with remaining budget

### CP-2: Connect YouTube
- **As a** clipper,
- **I want to** connect my YouTube channel
- **So that** I can access videos for clipping

### CP-3: Fetch Videos
- **As a** clipper,
- **I want to** browse videos from a campaign's source
- **So that** I can choose what to clip

### CP-4: AI Clip (Highlights)
- **As a** clipper,
- **I want the** AI to identify highlights automatically
- **So that** I don't have to manually find engaging moments

### CP-5: AI Clip (Captions)
- **As a** clipper,
- **I want the** AI to add auto-captions
- **So that** my clips are engaging and accessible

### CP-6: Post to TikTok
- **As a** clipper,
- **I want to** post clips directly to TikTok
- **So that** I can grow my TikTok presence

### CP-7: Post to Reels/Shorts
- **As a** clipper,
- **I want to** post clips to Instagram Reels and YouTube Shorts
- **So that** I can reach multiple platforms

### CP-8: Submit to Campaign
- **As a** clipper,
- **I want to** submit my clip to the campaign
- **So that** I can get paid for views

### CP-9: Track Earnings
- **As a** clipper,
- **I want to** see my earnings from all campaigns
- **So that** I can track my income

### CP-10: Track Views
- **As a** clipper,
- **I want to** see how many views my clips get
- **So that** I can improve my content strategy

---

## Summary

| User Type | Stories |
|-----------|---------|
| **Campaign Creator** | 4 (create campaign, set budget, review, track) |
| **Clipper** | 10 (find campaigns, connect, clip, post, submit, earn) |
