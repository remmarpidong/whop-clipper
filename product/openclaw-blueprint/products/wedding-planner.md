## 1. Budget Tracker

### Features:
- Set total budget
- Auto-calculate remaining
- Track payments by vendor
- Category breakdown

### Categories:
| Category | Typical % |
|----------|-----------|
| Venue | 30-35% |
| Catering | 25-30% |
| Photography | 10-12% |
| Attire | 8-10% |
| Flowers | 8-10% |
| Music | 8-10% |
| Other | 10-15% |

### Formulas:
```
Total Budget: ${{user_input}}
Spent: =SUM(Vendors.Amount)
Remaining: =Total - Spent
Paid: =SUMIF(Vendors.Status="Paid")
Unpaid: =SUMIF(Vendors.Status="Pending")
```
