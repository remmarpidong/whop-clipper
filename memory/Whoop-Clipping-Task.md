# Whoop Clipping Task - Project Research

## What is Whoop?

WHOOP is a fitness wearable that tracks:
- **Sleep** - Sleep stages, quality, duration
- **Strain** - Daily strain score (0-21)
- **Recovery** - Body's readiness (0-100%)
- **Heart health** - HRV, ECG, blood pressure
- **Women's health** - Menstrual cycle tracking

## Developer Platform

Whoop has an **API for developers**:
- OAuth 2.0 authentication
- Webhooks for real-time events
- Access to health & fitness data
