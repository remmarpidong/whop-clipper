## 🎯 HOW THIS WORKS WITH OPENCLAW

Each agent = One OpenClaw sub-agent with:
- Unique system prompt (role + responsibilities)
- Access to specific tools/APIs
- Memory of their domain
- Communication channel (chat/DM)

Business owner chats with department → routes to specific agent → agent takes action → reports back.
