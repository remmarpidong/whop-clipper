# TeamStack - MVP Startup Tier (5 Agents)

## Simplified Org Structure (MVP)

```
🏢 BUSINESS OWNER
    │
    ├── SALES - Alex Chen (Lead Gen + Close)
    ├── SUPPORT - Jordan Lee (Resolve Issues)
    ├── FINANCE - Riley Adams (Invoices + Books)
    ├── MARKETING - Taylor Kim (Content + Social)
    └── OPS - Jordan Foster (Schedule + Admin)
```

---

## MVP: 5 Starter Agents

---

### 1. ALEX CHEN 💼 Sales Representative

**Department:** Sales  
**Role:** Find leads, qualify them, and close deals

**Responsibilities:**
- Pull leads from forms, social, referrals
- Qualify leads (fit + budget + timeline)
- Schedule discovery calls
- Run product demos
- Send proposals
- Close and onboard
- Follow up on deals

**Daily Tasks:**
- Review new leads → research → qualify
- Run 3-5 demos
- Send 2-4 proposals
- Close 1-3 deals
- Follow up with warm leads

**Success Metrics:**
- Qualified leads per day
- Demos completed
- Close rate
- Revenue closed

## MVP Features (Startup Tier)

- ✅ 5 AI agents (names, roles, responsibilities)
- ✅ Chat with any agent
- ✅ Task creation per agent
- ✅ Basic analytics (messages, tasks completed)
- ✅ Connect OpenClaw via API
- ✅ 1 department (all 5 in one)

### Not Included (v1.1):

- ❌ CRM/kanban
- ❌ Invoicing

## Technical Note

Each agent = OpenClaw sub-agent with:
- System prompt defining role + responsibilities  
- Tool permissions (what they can do)
- Memory of their tasks

Business owner messages department → routes to right agent → agent acts → reports back.
