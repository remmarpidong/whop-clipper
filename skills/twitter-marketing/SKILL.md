# Twitter/X Marketing Skill

Automatically post your digital products to Twitter/X.

## Setup

1. Get Twitter API access:
   - Apply at https://developer.twitter.com
   - Create a project with "Read and Write" permissions
   - Get API Key, API Secret, Access Token, Access Token Secret

2. Set environment variables:
   - `TWITTER_API_KEY`
   - `TWITTER_API_SECRET`
   - `TWITTER_ACCESS_TOKEN`
   - `TWITTER_ACCESS_TOKEN_SECRET`

Or use the xurl skill for OAuth-based authentication.

## Usage

Post a tweet:
```
Tweet about my AI Prompts bundle
```

Post with image:
```
Post this with image [attach file]: Check out my new AI Prompts bundle!
```

Reply to tweet:
```
Reply to @user with this message...
```

Search tweets:
```
Search for "AI prompts" and show me the results
```

## Best Practices

1. **Use threads** - Break longer content into numbered tweets
2. **Add images** - Posts with images get 2x engagement
3. **Include hashtags** - But keep it to 2-3 relevant ones
4. **Engage first** - Like and reply before posting
5. **Best times** - 9am, 12pm, 5pm EST
6. **Consistency** - Post regularly to grow following

## Engagement Commands
