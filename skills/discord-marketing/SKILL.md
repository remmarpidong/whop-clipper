# Discord Marketing Skill

Share your digital products in Discord communities.

## Setup

1. The skill uses OpenClaw's existing Discord channel
2. Make sure Discord is configured in your OpenClaw config
3. Get invite links to target servers

## Usage

Share in a channel:
```
Share my AI Prompts in #marketing
```

Send to multiple channels:
```
Post this to #promotions and #general
```

Send DM:
```
Send this to user#1234: Check out my new product!
```

## Example Messages

**Product Post:**
```
🎉 New Product Alert!

🔥 500+ AI Prompts Bundle

Your AI assistant just got smarter.
Get proven prompts for marketing, sales, content & more.

👉 [link]

#AI #Productivity
```

**Engagement Post:**
```
🤔 What prompts do you use most?

I'm curious what AI prompts work best for this community!

Drop your favorites below 👇

#AI #ChatGPT #Prompts
```

**Value Post:**
```
💡 Quick Tip: Use this prompt to write better emails

"Write a follow-up email to [client] about [project]. 
Keep it professional but friendly. 
Include [specific request]."

Want 500+ more? Check out my bundle 👉 [link]

#EmailMarketing #Productivity
```

## Best Practices

1. **Read the room** - Don't spam, provide value first
2. **Follow rules** - Each server has promotion rules
3. **Be helpful** - Answer questions, contribute
4. **Use threads** - Keep conversations organized
5. **Engage naturally** - Don't just drop links

## Target Servers

**Productivity/AI:**
- Notion community servers
- AI & ChatGPT servers
- Productivity hubs

**Business:**
- Freelance communities
- Digital marketing groups
- Entrepreneur forums

**Tech:**
- OpenClaw Discord
- Developer communities
- SaaS groups

## Tips

- Lead with value, then mention product
- Use Discord embeds for professional look
- Add reactions to increase visibility
- Time posts for peak activity hours
